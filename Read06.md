# A review About Css

1. ***Definition***
   ***Cascading Style Sheets allows you to create great-looking web pages.***

2. ***Css syntax:***

 [h1 {
    color: ;
}.]
  
   ***How to add css in HTML file***

* *external css*
  
      [link rel="stylesheet" href="my style.css"]

* *inline css*

       *<h1 style="color:;text-align:;"></h1>*
  
* *internal css*

       [style]

       body {
       background-color: linen;
       }

         h1 {
         color: maroon;
         margin-left: 40px;
         }

       [/style]
